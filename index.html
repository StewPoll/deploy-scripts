Before running these scripts, please ensure the following:

  • You have SSH access to your Ubuntu 16.04 server
  • Your User on the server is sudo non root
  • You have opened port 80
  • You have opened port 443

Install LEMP Stack

This script installs a LEMP Stack, ready to deply wordpress sites with later on. From testing, this script works wonderufll 
curl https://scripts.stewpolley.com/lemp-setup.sh | sh
Please make sure to save the root MYSQL password somewhere secure when finished!

Install Wordpress

This script will ask you for the following - please have these ready first!

  • MYSQL Root Password (from lemp stack installation)
  • Domains for the new wordpress install
  • Directory under /var/www/html/ you would like wordpress installed

The final step in this script is installing a Let's Encrypt Certificate. This will require user interaction

curl https://scripts.stewpolley.com/install-wordpress.sh >> install-wordpress.sh && sh install-wordpress.sh
