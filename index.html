Deploy Scripts

By StewPolley

Just a collection of scripts to help get you deployed on Digital Ocean or Lightsail etc, just a little easier.

Before running these scripts, please ensure the following:

  • You have SSH access to your Server
  • Your User on the server is sudo non root
  • You have opened port 80
  • You have opened port 443

Lemp Stack

Install LEMP Stack

This script installs a LEMP Stack, ready to deply wordpress sites with later on. Comes with Ubuntu 16.04 and Ubuntu 18.04

Ubuntu Version Command
16.04 curl https://scripts.stewpolley.com/ubuntu-16-04/lemp/initial-setup.sh | sh
18.04 curl https://scripts.stewpolley.com/ubuntu-18-04/lemp/initial-setup.sh | sh

Please make sure to save the root MYSQL password somewhere secure when finished!

Install Wordpress

This script will ask you for the following - please have these ready first!

  • MYSQL Root Password (from lemp stack installation)
  • Domains for the new wordpress install
  • Directory under /var/www/html/ you would like wordpress installed

The final step in this script is installing a Let's Encrypt Certificate. This will require user interaction

Ubuntu Version Command
16.04 curl https://scripts.stewpolley.com/ubuntu-16-04/lemp/install-wordpress.sh >> install-wordpress.sh && sh install-wordpress.sh
18.04 curl https://scripts.stewpolley.com/ubuntu-18-04/lemp/install-wordpress.sh >> install-wordpress.sh && sh install-wordpress.sh
Once this script has run, visit a domain to ensure setup is finalised as this process currently does not setup the admin username/password
